# Product Overview

Snake Game — a browser-based implementation of the classic Snake arcade game built as a single-page React application.

## Core Gameplay
- Player controls a snake on a 20×20 grid using arrow keys, swipe gestures, or on-screen mobile buttons
- Snake grows by eating food; score increments per food consumed
- Game ends on wall collision or self-collision
- Adjustable speed slider (1–10) affects tick interval
- Pause/resume via spacebar or button
- High score persisted in localStorage

## Additional Features
- Internationalization: 6 languages (en, vi, de, zh, fr, ja)
- Retro background music via Web Audio API (opt-in, muted by default)
- Dark-themed neon visual style with particle background and glow effects
- Mobile-responsive with touch swipe controls and on-screen D-pad
- Game over dialog with score summary
